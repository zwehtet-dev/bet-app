# PWA Icons

Place your app icons here with the following sizes:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

You can generate these from your logo using tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

For best results, use a square image with transparent background.
