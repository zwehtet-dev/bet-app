# Background Images

Place your background images in this directory.

## Required Files:
- `bg-1.jpeg` - Main background image for the app

The layout is configured to use `/images/bg-1.jpeg` as the background image.

## Current Setup:
- Background image: `/images/bg-1.jpeg`
- Background size: cover (fills entire container)
- Background position: center
- Background repeat: no-repeat
- Overlay: Gradient overlay for better text readability

## Image Specifications:
- Recommended size: 430px width (mobile optimized)
- Format: JPEG for better compression
- Aspect ratio: Portrait orientation preferred for mobile
